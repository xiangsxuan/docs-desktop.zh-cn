# <a name="using-regions"></a>使用区域
GDI + <xref:System.Drawing.Region> 类允许您定义自定义形状。 该形状可以由直线、多边形和曲线组成。  
  
 区域的两个常见用途是命中测试和剪裁。 命中测试正在确定是否在屏幕的某个区域中单击鼠标。 剪辑正在将绘图限制为特定区域。
