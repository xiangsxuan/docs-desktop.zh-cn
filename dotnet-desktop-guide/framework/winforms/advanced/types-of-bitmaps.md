 <span data-ttu-id="438c0-187">PNG 在 GIF 中的改进功能可以逐渐显示图像 (也就是说，在通过网络连接) 时，可更好、更好地将图像更近似地显示。</span><span class="sxs-lookup"><span data-stu-id="438c0-187">PNG improves on GIF in its ability to progressively display an image (that is, to display better and better approximations of the image as it arrives over a network connection).</span></span> <span data-ttu-id="438c0-188">PNG 文件可以包含伽玛更正和颜色更正信息，以便能够在各种显示设备上准确呈现图像。</span><span class="sxs-lookup"><span data-stu-id="438c0-188">PNG files can contain gamma correction and color correction information so that the images can be accurately rendered on a variety of display devices.</span></span>  
  
### <a name="tag-image-file-format-tiff"></a><span data-ttu-id="438c0-189">标记图像文件格式 (TIFF) </span><span class="sxs-lookup"><span data-stu-id="438c0-189">Tag Image File Format (TIFF)</span></span>  
 <span data-ttu-id="438c0-190">TIFF 是一种灵活且可扩展的格式，由各种平台和图像处理应用程序支持。</span><span class="sxs-lookup"><span data-stu-id="438c0-190">TIFF is a flexible and extendable format that is supported by a wide variety of platforms and image-processing applications.</span></span> <span data-ttu-id="438c0-191">TIFF 文件可以存储具有任意数量的每像素位数的图像，并可以使用多种压缩算法。</span><span class="sxs-lookup"><span data-stu-id="438c0-191">TIFF files can store images with an arbitrary number of bits per pixel and can employ a variety of compression algorithms.</span></span> <span data-ttu-id="438c0-192">多个图像可以存储在一个多页 TIFF 文件中。</span><span class="sxs-lookup"><span data-stu-id="438c0-192">Several images can be stored in a single, multiple-page TIFF file.</span></span> <span data-ttu-id="438c0-193">与映像相关的信息 (扫描程序、主机计算机、压缩类型、方向、每个像素的样本等等，) 可以存储在文件中，并通过使用标记进行排列。</span><span class="sxs-lookup"><span data-stu-id="438c0-193">Information related to the image (scanner make, host computer, type of compression, orientation, samples per pixel, and so on) can be stored in the file and arranged through the use of tags.</span></span> <span data-ttu-id="438c0-194">TIFF 格式可以根据需要通过审批和添加新标记进行扩展。</span><span class="sxs-lookup"><span data-stu-id="438c0-194">The TIFF format can be extended as needed by the approval and addition of new tags.</span></span>
