# <a name="using-transformations-in-managed-gdi"></a>在托管 GDI+ 中使用变换
仿射转换包括旋转、缩放、反射、倾斜和平移。 在 GDI + 中， <xref:System.Drawing.Drawing2D.Matrix> 类提供对矢量绘图、图像和文本执行仿射转换的基础。
