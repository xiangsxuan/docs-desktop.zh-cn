# <a name="coordinate-systems-and-transformations"></a>坐标系和坐标转换
GDI + 提供了一个世界转换和一个页面转换，以便您可以转换 (旋转、缩放、平移等) 所绘制的项。 这两种转换还允许您在各种坐标系统中工作。
