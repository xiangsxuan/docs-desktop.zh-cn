# <a name="recoloring-images"></a>对图像重新着色
重新着色是调整图像颜色的过程。 重新着色的一些示例包括：将一种颜色更改为另一种颜色、调整颜色相对于另一种颜色的强度、调整所有颜色的亮度或对比度，以及将颜色转换为灰色阴影。
