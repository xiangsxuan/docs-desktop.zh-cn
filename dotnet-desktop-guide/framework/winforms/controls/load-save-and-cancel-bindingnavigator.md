# <a name="how-to-add-load-save-and-cancel-buttons-to-the-windows-forms-bindingnavigator-control"></a><span data-ttu-id="b6c59-103">如何：向 Windows 窗体 BindingNavigator 控件添加“加载”、“保存”和“取消”按钮</span><span class="sxs-lookup"><span data-stu-id="b6c59-103">How to: Add Load, Save, and Cancel Buttons to the Windows Forms BindingNavigator Control</span></span>

<span data-ttu-id="b6c59-104"><xref:System.Windows.Forms.BindingNavigator>控件是一种特殊用途的 <xref:System.Windows.Forms.ToolStrip> 控件，用于在窗体上导航和操作绑定到数据的控件。</span><span class="sxs-lookup"><span data-stu-id="b6c59-104">The <xref:System.Windows.Forms.BindingNavigator> control is a special-purpose <xref:System.Windows.Forms.ToolStrip> control that is intended for navigating and manipulating controls on your form that are bound to data.</span></span>

<span data-ttu-id="b6c59-105">由于它是一个 <xref:System.Windows.Forms.ToolStrip> 控件，因此 <xref:System.Windows.Forms.BindingNavigator> 可以轻松修改组件，以包含用户的其他或其他命令。</span><span class="sxs-lookup"><span data-stu-id="b6c59-105">Because it's a <xref:System.Windows.Forms.ToolStrip> control, the <xref:System.Windows.Forms.BindingNavigator> component can be easily modified to include additional or alternative commands for the user.</span></span>

## <a name="add-load-save-and-cancel-buttons-to-the-bindingnavigator-component"></a><span data-ttu-id="b6c59-107">向 BindingNavigator 组件添加 "加载"、"保存" 和 "取消" 按钮</span><span class="sxs-lookup"><span data-stu-id="b6c59-107">Add load, save, and cancel buttons to the BindingNavigator component</span></span>

5. <span data-ttu-id="b6c59-113">选择 <xref:System.Windows.Forms.BindingNavigator> 控件。</span><span class="sxs-lookup"><span data-stu-id="b6c59-113">Select the <xref:System.Windows.Forms.BindingNavigator> control.</span></span>

6. <span data-ttu-id="b6c59-114">单击 "设计器操作" 标志符号 (![ 小型黑色箭头 ](./media/designer-actions-glyph.gif)) ，此时将显示 " **BindingNavigator 任务** " 对话框，然后选择 " **编辑项目**"。</span><span class="sxs-lookup"><span data-stu-id="b6c59-114">Click the designer actions glyph (![Small black arrow](./media/designer-actions-glyph.gif)) so the **BindingNavigator Tasks** dialog appears and select **Edit Items**.</span></span>

     <span data-ttu-id="b6c59-115">**项集合编辑器** 随即出现。</span><span class="sxs-lookup"><span data-stu-id="b6c59-115">The **Items Collection Editor** appears.</span></span>

7. <span data-ttu-id="b6c59-116">在 **Items 集合编辑器** 中，完成以下操作：</span><span class="sxs-lookup"><span data-stu-id="b6c59-116">In the **Items Collection Editor**, complete the following:</span></span>

    1. <span data-ttu-id="b6c59-117"><xref:System.Windows.Forms.ToolStripSeparator> <xref:System.Windows.Forms.ToolStripButton> 通过选择适当的类型 <xref:System.Windows.Forms.ToolStripItem> 并单击 "**添加**" 按钮，添加一个和三个项。</span><span class="sxs-lookup"><span data-stu-id="b6c59-117">Add a <xref:System.Windows.Forms.ToolStripSeparator> and three <xref:System.Windows.Forms.ToolStripButton> items by selecting the appropriate type of <xref:System.Windows.Forms.ToolStripItem> and clicking the **Add** button.</span></span>

    2. <span data-ttu-id="b6c59-118">分别将 <xref:System.Windows.Forms.ToolStripItem.Name%2A> 按钮的属性设置为 **LoadButton**、 **SaveButton** 和 **CancelButton**。</span><span class="sxs-lookup"><span data-stu-id="b6c59-118">Set the <xref:System.Windows.Forms.ToolStripItem.Name%2A> property of the buttons to **LoadButton**, **SaveButton**, and **CancelButton**, respectively.</span></span>

    3. <span data-ttu-id="b6c59-119">将 <xref:System.Windows.Forms.ToolStripItem.Text%2A> 按钮的属性设置为 " **加载**"、" **保存**" 和 " **取消**"。</span><span class="sxs-lookup"><span data-stu-id="b6c59-119">Set the <xref:System.Windows.Forms.ToolStripItem.Text%2A> property of the buttons to **Load**, **Save**, and **Cancel**.</span></span>

    4. <span data-ttu-id="b6c59-120">将 <xref:System.Windows.Forms.ToolStripItem.DisplayStyle%2A> 每个按钮的属性设置为 " **文本**"。</span><span class="sxs-lookup"><span data-stu-id="b6c59-120">Set the <xref:System.Windows.Forms.ToolStripItem.DisplayStyle%2A> property for each of the buttons to **Text**.</span></span> <span data-ttu-id="b6c59-121">或者，您可以将此属性设置为 **image** 或 **ImageAndText**，并将该图像设置为显示在 <xref:System.Windows.Forms.ToolStripItem.Image%2A> 属性中。</span><span class="sxs-lookup"><span data-stu-id="b6c59-121">Alternatively, you can set this property to **Image** or **ImageAndText**, and set the image to be displayed in the <xref:System.Windows.Forms.ToolStripItem.Image%2A> property.</span></span>

    5. <span data-ttu-id="b6c59-122">单击“确定”  关闭对话框。</span><span class="sxs-lookup"><span data-stu-id="b6c59-122">Click **OK** to close the dialog box.</span></span> <span data-ttu-id="b6c59-123">这些按钮将添加到 <xref:System.Windows.Forms.ToolStrip> 。</span><span class="sxs-lookup"><span data-stu-id="b6c59-123">The buttons are added to the <xref:System.Windows.Forms.ToolStrip>.</span></span>

8. <span data-ttu-id="b6c59-124">右键单击窗体，然后选择 " **查看代码**"。</span><span class="sxs-lookup"><span data-stu-id="b6c59-124">Right-click the form and choose **View Code**.</span></span>

10. <span data-ttu-id="b6c59-129">为之前创建的负载的事件创建事件处理程序 <xref:System.Windows.Forms.ToolStripItem.Click>  <xref:System.Windows.Forms.ToolStripButton> ，并将此数据加载到其中。</span><span class="sxs-lookup"><span data-stu-id="b6c59-129">Create an event handler for the <xref:System.Windows.Forms.ToolStripItem.Click> event of the **Load** <xref:System.Windows.Forms.ToolStripButton> you created earlier and move this data-loading code into it.</span></span>

     <span data-ttu-id="b6c59-130">你的代码现在应如下所示：</span><span class="sxs-lookup"><span data-stu-id="b6c59-130">Your code should now look similar to the following:</span></span>

    ```vb
        TableAdapterName.Fill(DataSetName.TableName)

    ```
