# <a name="tabcontrol-control-windows-forms"></a>TabControl 控件（Windows 窗体）
Windows 窗体 `TabControl` 显示多个选项卡，就像笔记本中的分隔线或档案柜中一组文件夹的标签。 选项卡可以包含图片和其他控件。 使用 `TabControl` 创建属性页。
