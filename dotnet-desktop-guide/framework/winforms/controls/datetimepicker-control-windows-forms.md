# <a name="datetimepicker-control-windows-forms"></a>DateTimePicker 控件（Windows 窗体）
Windows 窗体 `DateTimePicker` 控件允许用户从日期或时间列表中选择单个项。 用于表示日期时，它显示在两个部分：一个下拉列表，其中包含以文本表示的日期，当您单击列表旁边的向下箭头时，将显示网格。
