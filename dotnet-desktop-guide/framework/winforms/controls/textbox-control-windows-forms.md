# <a name="textbox-control-windows-forms"></a>TextBox 控件（Windows 窗体）
Windows 窗体文本框用于获取用户的输入或显示文本。 `TextBox`控件通常用于可编辑文本，但也可将其设为只读。 文本框可显示多个行，将文本自动换行到控件大小，并添加基本格式设置。 `TextBox`控件对于显示或输入到控件中的文本允许使用一种格式。
