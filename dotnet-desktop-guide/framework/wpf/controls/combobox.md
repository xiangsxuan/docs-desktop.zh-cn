# <a name="combobox"></a>组合框
<xref:System.Windows.Controls.ComboBox>控件向用户提供选项列表。 当控件展开和折叠时，列表将显示并隐藏。 在其默认状态下，列表将折叠，只显示一个选项。 用户单击按钮以查看选项的完整列表。
