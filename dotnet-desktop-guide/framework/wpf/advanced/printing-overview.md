## <a name="xpsdrv-driver-model"></a><span data-ttu-id="521ea-194">XPSDrv 驱动程序模型</span><span class="sxs-lookup"><span data-stu-id="521ea-194">XPSDrv Driver Model</span></span>

 <span data-ttu-id="521ea-198">EMF 是一种封闭格式，它将应用程序输出表示为对用于呈现服务的 GDI 进行的一系列调用。</span><span class="sxs-lookup"><span data-stu-id="521ea-198">EMF is a closed format that represents application output as a series of calls into GDI for rendering services.</span></span> <span data-ttu-id="521ea-199">与 EMF 不同，XPS 假脱机格式代表实际的文档，而在输出到基于 XPS 的打印机驱动程序 (XPSDrv) 时无需进一步解释。</span><span class="sxs-lookup"><span data-stu-id="521ea-199">Unlike EMF, the XPS spool format represents the actual document without requiring further interpretation when output to an XPS-based printer driver (XPSDrv).</span></span> <span data-ttu-id="521ea-200">这些驱动程序可以用这种格式直接对数据进行操作。</span><span class="sxs-lookup"><span data-stu-id="521ea-200">The drivers can operate directly on the data in the format.</span></span> <span data-ttu-id="521ea-201">此功能消除了使用 EMF 文件和基于 GDI 的打印驱动程序时所需的数据和颜色空间转换。</span><span class="sxs-lookup"><span data-stu-id="521ea-201">This capability eliminates the data and color space conversions required when you use EMF files and GDI-based print drivers.</span></span>
