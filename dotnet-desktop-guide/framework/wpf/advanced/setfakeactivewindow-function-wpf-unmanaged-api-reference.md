此 API 支持 Windows Presentation Foundation (WPF) 基础结构，不应在代码中直接使用。  
  
 由用于 Windows 管理的 Windows Presentation Foundation (WPF) 基础结构使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
void __stdcall SetFakeActiveWindow(  
   HWND hwnd  
)  
```  
  
## <a name="parameters"></a>参数
