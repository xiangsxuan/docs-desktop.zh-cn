# <a name="xaml-in-wpf"></a>WPF 中的 XAML

[!INCLUDE[TLA#tla_xaml](../../../includes/tlasharptla-xaml-md.md)] 是用于声明性应用程序编程的标记语言。 [!INCLUDE[TLA#tla_winclient](../../../includes/tlasharptla-winclient-md.md)] 实现 XAML 处理器实现并提供 XAML 语言支持。 [!INCLUDE[TLA2#tla_winclient](../../../includes/tla2sharptla-winclient-md.md)]实现这些类型，以便它们可以为 XAML 表示形式提供所需的类型支持。 通常，可以 [!INCLUDE[TLA2#tla_winclient](../../../includes/tla2sharptla-winclient-md.md)] 在 XAML 标记中创建大多数应用程序 UI。
